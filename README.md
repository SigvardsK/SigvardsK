### Principles

```
1. Truth        — first principles, verified facts
2. Velocity     — when reversible; otherwise Impact
3. Impact       — magnitude × likelihood
4. Empathy      — people consequences, dignity preserved
5. Aesthetics   — clarity as elegance
```
